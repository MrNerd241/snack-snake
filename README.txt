EN - English
Hello! And welcome to my game! Snack Snake.

Warning!
1. Make sure python is installed on your pc!
2. It is forbidden to change the code in main.py

How to install python:
Option 1. Open www.python.org
Option 2. Open the "pythons" folder, and select one of the files according to your OS

How to play:
1. Open cmd file then write in cmd "pip install pygame" and wait for the installation
2. Open "main.py" to play the game
3. Have fun! :D

Control:
W / ↑ = Forward
S / ↓ = Backward
A / ← = Turn left
D / → = Turn right
