EN - English
Hello! And welcome to my game! Snack Snake.

Warning!
1. Make sure Python is installed on your PC!
2. It is forbidden to change the code in main.py

How to install python:
Option 1:
  1. Go to www.python.org
  2. Find the latest python version and your OS
Option 2: 
  1. Open Microsoft Store
  2. Wait
How to play:
1. Open the cmd file then write in cmd "pip install pygame" and wait for the installation
2. Open "main.py" to play the game
3. Have fun! :D

Control:
W / ↑ = Forward
S / ↓ = Backward
A / ← = Turn left
D / → = Turn right
